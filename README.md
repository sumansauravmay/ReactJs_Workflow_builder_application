## Run Locally

Clone the project

```bash
  git clone https://github.com/sumansauravmay/ReactJs_Workflow_builder_application.git
```

Go to the project directory

```bash
  cd tech_stax

```

Install dependencies for tech_stax as well as backend

```bash
  cd tech_stax
  npm install
```

```bash
  cd Backend
  npm install
```

Start the localhost server

```bash
  cd tech_stax
  npm start
```
